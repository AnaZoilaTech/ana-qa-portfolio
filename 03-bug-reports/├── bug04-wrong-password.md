### BUG-04 – Missing Error Message on Wrong Password

**Severity:** Critical  

**Description:**  
When entering a wrong password, the login page does not display an error message.

**Steps to Reproduce:**  
1. Enter a valid email  
2. Enter an incorrect password  
3. Click “Login”  

**Expected Result:**  
User should see an error message like “Incorrect password”.

**Actual Result:**  
No error message is shown.

**Evidence:**  
"Wrong password – no UI feedback"
